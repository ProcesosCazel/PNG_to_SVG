# Convertidor de png a svg
Script en Python para convertir imágenes PNG en archivos SVG vectoriales mediante detección de contornos y simplificación geométrica.

## Ideal para:
- Logos
- Iconos
- Elementos gráficos simples

## Requisitos:
- Python 3.10+
    - (https://www.python.org/downloads/)
- Dependencias de py (instalar usando la terminal)
    - (pip install pillow numpy scikit-image)
- Imagen formato png
    - (Nombre del archivo a convertir: "Imagen.png")

## Como ejecutar:
1. Coloca el script de python y la imagen a convertir en una misma carpeta.
2. Abrir terminal (Símbolo del sistema)
3. Escribir "cd" y la ruta a la carpeta donde está el script, por ejemplo:
    - cd C:\Users\procesos.CAZEL\Desktop\Becario Procesos\PNG to SVG Converter\
4. Escribir "py" y el nombre del archivo del script de python, por ejemplo:
    - py png_to_svg.py
